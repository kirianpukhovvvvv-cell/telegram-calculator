# Калькулятор Telegram‑бота с конвертацией

## Основные функции

### Математические операции
* Базовые: `5 + 3`, `10 - 4`, `6 * 7`, `15 / 3`
* Дроби: `1/2 + 1/4` → `3/4`
* Корни: `sqrt(16)` → `4.0`
* Тригонометрия: `sin(30)` → `0.5`, `cos(60)` → `0.5`

### Конвертация величин

**Физические единицы:**
* Длина: `2 м в см` → `2 м = 200 см`
* Вес: `1.5 кг в г` → `1.5 кг = 1500 г`

**Валюты:**
* `100 USD в EUR` — конвертация по актуальному курсу
* Поддерживаемые валюты: USD, EUR, RUB, GBP, JPY и др.

### Русские команды
Можно использовать русские слова:
* `5 плюс 3` вместо `5 + 3`
* `корень 16` вместо `sqrt(16)`
* `синус 30` вместо `sin(30)`
* `2 умножить на 5` вместо `2 * 5`

## Примеры использования

1. **Математика:**
   * `2 + 3 * 4` → `14`
   * `1/2 + 1/3` → `5/6`
   * `sqrt(25)` → `5.0`
   * `sin(90)` → `1.0`

2. **Конвертация:**
   * `3 м в см` → `3 м = 300 см`
   * `0.5 кг в г` → `0.5 кг = 500 г`
   * `50 USD в RUB` → `50 USD = 4500.00 RUB`

3. **Русские команды:**
   * `5 плюс корень 9` → `8.0`
   * `синус 30` → `0.
