# Telegram Calculator Bot

Бот‑калькулятор для Telegram с поддержкой арифметических и тригонометрических операций.

## Функционал

- Арифметические операции: `+`, `-`, `*`, `/`
- Тригонометрические функции: `sin`, `cos`, `tg`, `ctg`
- Квадратный корень: `sqrt`
- Работа с дробями: `Fraction(числитель, знаменатель)`
- Поддержка скобок и сложных выражений
- Вывод результата в виде дроби и десятичного числа
