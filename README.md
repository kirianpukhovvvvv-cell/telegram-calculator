# Telegram Calculator Bot

Бот‑калькулятор для Telegram с поддержкой арифметических и тригонометрических операций на базе библиотеки sympy.

## Функционал

- Арифметические операции: `+`, `-`, `*`, `/`
- Тригонометрические функции: `sin`, `cos`, `tg`, `ctg`
- Квадратный корень: `sqrt`
- Работа с дробями: `1/2`, `3/4` и т. д.
- Константы: `pi`, `e`
- Поддержка скобок и сложных выражений
- Вывод результата в виде дроби, десятичного и точного значения
