# Калькулятор Telegram‑бота

## Доступные операции

### Базовые математические операции
* Сложение: `5 + 3`
* Вычитание: `10 - 4`
* Умножение: `6 * 7`
* Деление: `15 / 3`

### Дроби
* Обыкновенные дроби: `1/2 + 1/4` (результат: `3/4`)
* Смешанные операции: `2 + 1/2` (результат: `5/2`)

### Квадратный корень
* `sqrt(16)` → `4.0`
* `sqrt(2)` → `1.414214`

### Тригонометрические функции (углы в градусах)
* Синус: `sin(30)` → `0.5`
* Косинус: `cos(60)` → `0.5`
* Тангенс: `tan(45)` → `1.0`
* Котангенс: `ctg(45)` → `1.0`

## Примеры использования

1. `2 + 3 * 4` → `14`
2. `1/2 + 1/3` → `5/6`
3. `sqrt(25)` → `5.0`
4. `sin(90)` → `1.0`
5. `cos(0)` → `1.0`
6. `tan(45) + 1` → `2.0`
7. `1/4 * 8` → `2`

## Важные замечания
* Все тригонометрические функции принимают углы в градусах.
* Для дробей используйте формат `числитель/знаменатель`.
* Поддерживаются отрицательные числа: `-5 + 3`, `sin(-30)`.
* Используйте скобки для сложных выражений: `(2 + 3) * 4`.
