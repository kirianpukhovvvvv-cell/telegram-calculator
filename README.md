# Калькулятор Telegram‑бота с конвертацией

## Новые возможности

### Русские команды
Теперь можно использовать:

**Математические термины:**
* `корень 100` вместо `sqrt(100)`
* `синус 30` вместо `sin(30)`
* `косинус 60` вместо `cos(60)`
* `тангенс 45` вместо `tan(45)`
* `котангенс 45` вместо `ctg(45)`

**Числа русскими словами (полноценная поддержка от нуля до секстиллионов):**
* **Единицы:** `ноль`, `один`, `два`, ..., `девять`
* **Десятки:** `десять`, `одиннадцать`, ..., `девятнадцать`, `двадцать`, `тридцать`, ..., `девяносто`
* **Сотни:** `сто`, `двести`, ..., `девятьсот`
* **Масштабные единицы:** `тысяча`, `миллион`, `миллиард`, `триллион`, `квадриллион`, `квинтиллион`, `секстиллион` и их формы (`тысячи`, `миллионов` и т. д.)

### Примеры использования

**Математика с русскими терминами:**
* `корень сто` → `10.0`
* `синус тридцать` → `0.5`
* `косинус шестьдесят` → `0.5`
* `тангенс сорок пять` → `1.0`

**Числа русскими словами:**
* Простые числа:
  * `двадцать пять` → `25`
  * `сто пятьдесят три` → `153`
* С масштабными единицами:
  * `две тысячи` → `2000`
  * `пять миллионов` → `5000000`
  * `три миллиарда` → `3000000000`
  * `одна тысяча двести` → `1200`
  * `два миллиона пятьсот тысяч` → `2500000`
* Комбинированные:
  * `три миллиона двести тысяч сто пятьдесят` → `3200150`
  * `пять миллиардов семьсот миллионов` → `5700000000`

**Конвертация величин:**
* Длина: `два метра в сантиметры` → `2 м = 200 см`
* Вес: `полтора килограмма в граммы` → `1.5 кг = 1500 г`

**Калькулятор валют:**
* `сто долларов в евро` → `100 USD = ... EUR` (актуальный курс)
* `пятьдесят евро в рубли` → `50 EUR = ... RUB` (актуальный курс)

**Комбинированные примеры:**
* `корень из ста` → `10.0` (поддерживается предлог «из»)
* `два плюс три умножить на десять` → `32`
* `пятьсот разделить на двадцать пять` → `20`
* `синус сорока пяти градусов` → `0.707107`

## Полный синтаксис

**Поддерживаемые операции:**
* Арифметика: `+`, `-`, `*`, `/` (или русские: `плюс`, `минус`, `умножить`, `разделить`)
* Функции: `sqrt()` (`корень`), `sin()` (`синус`), `cos()` (`косинус`), `tan()` (`тангенс`), `ctg()` (`котангенс`)
* Конвертация: `[число] [единица1] в [единица2]`
* Валюты: `[сумма] [код1] в [код2]` (USD, EUR, RUB и др.)
* Числа: любые комбинации русских слов от `ноль` до `секстиллион`
